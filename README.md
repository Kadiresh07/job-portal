- Role Based Access
- Company Management
- Job Management
- Job Application
- Employer Dashboard
- Candidate Dashboard
- Admin Dashboard
- Resume Upload

## Tech Stack

Frontend

- Next.js
- React
- Tailwind CSS

Backend

- Node.js
- Express.js

Database

- MongoDB Atlas

Deployment

- Vercel
- Render

## Installation

### Clone

```bash
git clone https://github.com/Kadiresh07/job-portal.git
```

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
